09/2023 – Hiện tại

Team Leader | CÔNG TY CỔ PHẦN KIDSONLINE

  • Chủ trì rebuild hệ thống Kidsonline theo mô hình Domain-Driven Design (DDD) và kiến trúc Microservices, **nâng cao hiệu suất 25% và giảm thiểu lỗi 15%**.
  • Dẫn dắt đội nhóm 3-4 thành viên, **thiết lập quy trình làm việc Agile/SCRUM hiệu quả, tăng năng suất team lên 20%**. Đảm bảo chất lượng code thông qua code review và mentorship.
  • Dẫn dắt dự án tái cấu trúc API APP phụ huynh Kidsonline (800.000 người dùng, 3 triệu requests/ngày), **cải thiện tốc độ phản hồi API trung bình 30%**.
  • Tiên phong áp dụng AI (AWS facial recognition, Hanet camera API) tự động hóa điểm danh, **giảm 30% thời gian nhận xét hoạt động học sinh cho giáo viên**.
  • Thiết kế và triển khai hệ thống quản lý tài chính tích hợp (DDD), **tối ưu quy trình thu chi, giảm 40% công sức thủ công** và tích hợp thanh toán (VietQR, Vietinbank, Gpay).
